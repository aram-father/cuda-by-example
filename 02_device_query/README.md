# 02_device_query

**Author:** Wonseok Lee (aram_fahter@naver.com)

**Last update:** 2021-03-09(TUE)

## What does it do?

Print device properties of installed GPUs.

## Usage

```bash
./02_device_query
# Expected output
# Device Index: 0
#  - name                       : GeForce MX150
#  - totalGlobalMem             : 2099904512 bytes
#  - sharedMemPerBlock          : 49152 bytes
#  - regsPerBlock               : 65536
#  ...
```
